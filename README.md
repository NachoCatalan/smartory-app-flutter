# Smartory-App Mobile

_Aplicación móvil enfocada en la gestión inteligente de inventario del hogar, permitiendo registrar, visualizar y administrar productos, con proyección a integrar entrada por voz e inteligencia artificial._

# Dev

1. Instalar dependencias

2. En la consola correr `flutter doctor` y configurar lo necesario que falte


3. Ejecutar la app `flutter run` o **F5 en main.dart**


---

## Implementaciones realizadas

 * Toma de notas de audio y envio a backend para procesamiento.
 * Recepcion de instrucciones procesadas.
 * Listado de productos con detalle
 * Navegación básica por Stacks
